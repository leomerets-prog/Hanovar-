# Hanovar

Site institucional e vitrine da Hanovar — onde vestimos grandes mulheres.

## Estrutura

- `index.html` — site completo (HTML + CSS + JS em um único arquivo)
- `images/` — fotos do site

## Como rodar localmente

Basta abrir o `index.html` no navegador. Ou, com Python instalado:

```bash
python3 -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Hospedagem

Hospedado na Vercel. O deploy é automático a cada `git push` na branch `main`.
